# Higher Or Lower Game

2 Versions Main Game & Debug Version 

1. Main Game
   - Open run.bat
   - Select Difficulty
   - Enter your guess and have fun!
   - The menu should pop up and pick your option! 

2. Debug Verison
   - Open debug.bat
   - Select Difficulty
   - The guess will come up and you can see the menu!
  
# Commands
R - Restart Game
L - View Logs
X - Close Game
